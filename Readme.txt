Proyecto: SOCIAL POST

Realizamos una pagina web en donde el usuario puede realizar la consulta de 10 usuarios pertenecientes a la red social y observar los post que este usuario a realizado, adicional al dar click sobre alguno de los post sera posible observar los comentarios que contiene dicho post.

Decidimos utilizar una api conocida como https://randomuser.me/api para obtener una imagen de usuario y brindar un ambiente similar al de una red social, adicional utilizamos la api https://rickandmortyapi.com/api/character/ con el fin de dar un contenido visual a cada post
